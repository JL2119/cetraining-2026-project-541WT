## 项目简介 
## 一个基于python的学生成绩管理系统，支持学生信息管理、成绩录入、成绩统计和查询功能。
## 核心功能列表
## 1.学生信息管理录入（增删改查）
## 2.成绩录入与修改
## 3.成绩统计（平均分、最高分、最低分）
## 4.成绩查询与排序
## 5.数据持久化（文件保存与读取）
